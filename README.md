# Nimbus - UE4 Project

Using the super dank UFSM plugin, this is a platformer game.

## Controls

| Key(s) | Effect |
| --- | --- |
| AD | Movement |
| Space | Jump |
| F | Normal Attack |
| E | Up Smash |
| R | Down Smash |
